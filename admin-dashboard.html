إدخال GitHub Token

يرجى إدخال GitHub Personal Access Token للمستودع لتتمكن من إجراء التعديلات.

كيفية إنشاء Token:
  1. اذهب إلى GitHub Settings → Tokens
  2. انقر "Generate new token (classic)"
  3. اختر صلاحية repo
  4. انسخ التوكن
