📋 كيفية الاختبار

⚠️ تعليمات مهمة

1. افتح صفحة index.html الرئيسية

2. تأكد من تسجيل الدخول كمطور (isDev = true)

3. ابحث عن زر 🎛️ في أسفل يمين الصفحة

4. اضغط على الزر لفتح لوحة التحكم الذكية

اختبار أوامر وحدة التحكم:

افتح وحدة التحكم (F12) واختبر هذه الأوامر:

// عرض الإعدادات الحالية showAudioConfig() // اختبار الموسيقى الخلفية بـ 5% setBackgroundMusicVolume(0.05) enableBackgroundMusic(true) // اختبار الموسيقى الخلفية بـ 1% setBackgroundMusicVolume(0.01) // اختبار موسيقى الصيانة بـ 5% setMaintenanceMusicVolume(0.05) enableMaintenanceMusic(true) // إيقاف الكل enableBackgroundMusic(false) enableMaintenanceMusic(false)
