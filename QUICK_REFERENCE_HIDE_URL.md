## 🚀 خطوات سريعة للنطاق المخصص

### 1. شراء النطاق
- اذهب إلى: Namecheap أو GoDaddy
- اشترِ النطاق المطلوب

### 2. إعداد DNS
في لوحة تحكم النطاق، أضف:
```
A Record:
@  →  185.199.108.153
@  →  185.199.109.153
@  →  185.199.110.153
@  →  185.199.111.153

CNAME Record:
www  →  aliabdelaal-adm.github.io
```

### 3. تفعيل في GitHub
```
1. Settings → Pages
2. Custom domain → أدخل النطاق
3. ✅ Enforce HTTPS
4. احفظ
```

### 4. الانتظار
```
⏰ 5-10 دقائق: DNS يتحدث
✅ بعدها: يعمل النطاق الجديد!
```
