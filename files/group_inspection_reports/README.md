# تقارير التفتيش الجماعي

هذا المجلد مخصص لتقارير التفتيش الجماعي المحملة من قبل المطور.

## الميزات:
- جميع المفتشين يمكنهم معاينة وتحميل التقارير
- المطور فقط يمكنه رفع التقارير من مصادر خارجية
- المطور فقط يمكنه حذف التقارير

## الملفات المدعومة:
- PDF
- Excel (.xlsx, .xls)
- Word (.docx, .doc)
- PowerPoint (.pptx, .ppt)
