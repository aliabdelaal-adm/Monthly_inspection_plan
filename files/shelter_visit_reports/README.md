# تقارير زيارة الملاجئ
## Shelter Visit Reports

هذا المجلد يحتوي على تقارير زيارة الملاجئ المرفوعة من قبل المطور.

---

### الملفات المدعومة:
- PDF (.pdf)
- Excel (.xlsx, .xls)
- Word (.docx, .doc)
- PowerPoint (.pptx, .ppt)

### كيفية رفع تقرير:
1. سجل الدخول كمطور
2. افتح قسم "تقارير زيارة الملاجئ"
3. استخدم قسم "رفع تقرير جديد"
4. اختر الملف وأدخل العنوان والوصف
5. اضغط "رفع التقرير"
