🔧 خيارات العرض والتصدير

  • عرض موحد: جدول واحد لجميع المحلات
  • عرض حسب المنطقة: جدول منفصل لكل منطقة
  • عرض حسب النشاط: تجميع المحلات حسب نوع النشاط
  • تصدير JSON: تصدير البيانات بصيغة JSON
  • طباعة: طباعة التقارير بتنسيق احترافي
  • إحصائيات شاملة: عرض إحصائيات مفصلة

📱 كيفية الاستخدام

  1. افتح أداة التخطيط الذكي (Smart Planner)
  2. انقر على زر "قائمة المحلات الذكية" 📋
  3. اختر طريقة الترتيب المفضلة (التقييم، المنطقة، النشاط، إلخ)
  4. استخدم الفلاتر لتصفية المحلات حسب المنطقة أو التقييم
  5. اختر طريقة العرض (موحد، حسب المنطقة، حسب النشاط)
  6. اطبع التقرير أو صدره حسب الحاجة
