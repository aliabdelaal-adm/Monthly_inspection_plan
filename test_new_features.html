✨ الميزات الجديدة المفعلة في Smart Planner

تم تفعيل جميع الميزات التي كانت قيد التطوير

🚀 فتح Smart Planner

✅ مفعل 🗺️ الخريطة التفاعلية والحرارية للمحلات

📍

عرض تفاعلي للمحلات

يمكنك الآن مشاهدة جميع المحلات على خريطة تفاعلية مع:

  • بطاقات ملونة حسب الأولوية (عادي، متوسط، عالي)
  • معلومات كاملة عند النقر على أي محل
  • تاريخ آخر تفتيش لكل محل
  • تصفية حسب المنطقة
🔥

الخريطة الحرارية (Heatmap)

عرض كثافة المحلات في كل منطقة مع:

  • ألوان متدرجة حسب عدد المحلات (أخضر → أحمر)
  • نسبة الكثافة لكل منطقة
  • إحصائيات فورية
🎯

عرض المجموعات (Cluster View)

تجميع المحلات حسب المنطقة مع عرض مرئي جذاب

📥

تصدير بيانات الخريطة

تصدير المحلات المعروضة على الخريطة بصيغة JSON

✅ مفعل 🔗 دمج بيانات المحلات

🔍

اكتشاف المحلات المكررة

يتم اكتشاف المحلات المكررة تلقائياً بناءً على:

  • التطابق التام في الأسماء
  • معاينة قبل الدمج مع تفاصيل كل محل مكرر
  • إحصائيات دقيقة عن عدد المكررات
🔤

اكتشاف المحلات المتشابهة

استخدام خوارزمية Levenshtein Distance للعثور على المحلات المتشابهة في الأسماء

  • نسبة تشابه قابلة للتخصيص (افتراضياً 70%)
  • عرض نسبة التشابه لكل زوج من المحلات
📥

دمج من ملف خارجي

استيراد محلات من ملف JSON ودمجها مع القائمة الحالية

  • دعم ملفات JSON
  • معاينة قبل الدمج
  • تجنب المكررات تلقائياً
💾

حفظ فوري في GitHub

جميع عمليات الدمج تُحفظ مباشرة في GitHub مع رسالة توضيحية

✅ مفعل ✏️ التعديل الجماعي للمحلات

☑️

تحديد متعدد للمحلات

يمكنك تحديد عدة محلات دفعة واحدة للتعديل

  • تحديد الكل / إلغاء تحديد الكل
  • تصفية حسب المنطقة
  • عداد للمحلات المحددة
🗺️

نقل جماعي للمناطق

نقل عدة محلات إلى منطقة جديدة دفعة واحدة

📝

إعادة تسمية جماعية

إضافة بادئة أو لاحقة لأسماء المحلات المحددة

  • مثال: إضافة "محل - " قبل الاسم
  • مثال: إضافة " - الفرع الرئيسي" بعد الاسم

✅ مفعل 🗑️ الحذف الجماعي للمحلات

⚠️

حذف آمن مع تأكيد مزدوج

نظام حماية متقدم لمنع الحذف العرضي

  • تحذير واضح قبل الحذف
  • تأكيد مزدوج للعملية
  • معاينة المحلات المحددة للحذف
🔍

تصفية قبل الحذف

إمكانية تصفية المحلات حسب المنطقة قبل الاختيار للحذف

📖 كيفية الاستخدام

  1. افتح Smart Planner من الزر أعلاه
  2. سجل الدخول بـ GitHub Token الخاص بك
  3. انتقل إلى تبويب "إدارة المحلات"
  4. ستجد جميع الأزرار الجديدة مفعلة وجاهزة للاستخدام:
    • 🗺️ عرض على الخريطة - لفتح الخريطة التفاعلية
    • 🔗 دمج بيانات - لدمج المحلات المكررة
    • ✏️ تعديل جماعي - لتعديل عدة محلات
    • 🗑️ حذف جماعي - لحذف عدة محلات

⚙️ التفاصيل التقنية

  • ✅ جميع العمليات تُحفظ مباشرة في GitHub
  • ✅ استخدام خوارزميات ذكية لاكتشاف التشابه (Levenshtein Distance)
  • ✅ واجهة مستخدم عربية كاملة وسهلة الاستخدام
  • ✅ معاينة قبل التنفيذ لجميع العمليات
  • ✅ نظام حماية متعدد المستويات
  • ✅ إحصائيات فورية ودقيقة
  • ✅ دعم كامل للتصدير والاستيراد
